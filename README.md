# De la Crête à la Mer — A Naturalist's Corsica

A family field dossier for a nature/biology-focused trip to southern Corsica
(Corse-du-Sud): five days at Bavella, then Porto-Vecchio. Endemic species,
an interactive schematic map, venues and activities, and fieldwork tips for a
curious young naturalist. Bilingual English / Čeština.

Single self-contained `index.html` (no build step, no external assets),
published with GitHub Pages.
